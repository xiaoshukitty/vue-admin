# 项目在线链接

[xiaoshukitty.github.io/vue_test/dist/#/](https://xiaoshukitty.github.io/vue_test/dist/#/)

## 项目启动	

cd 到 node_server目录，命令 nodemon app.js 启动服务，项目就可以启动了

## 文件位置

### 接口位置

接口在 node_server 文件中(node编写)

### 打包文件

在 dist 文件中

```js
// 打包之前 需要运行

npm run uncomment

// 用来把需要放开的注释用这个命令脚本来放开、在使用

npm run build
```
